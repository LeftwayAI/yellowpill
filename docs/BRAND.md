# Yellow Pill Brand Guidelines

## Colors

```css
--yellow-primary: #FCC800;     /* Brand yellow - use for CTAs, highlights */
--yellow-glow: rgba(252, 200, 0, 0.15);  /* Subtle glow effect */
--background: #000000;          /* Pure black - always */
--foreground: #FFFFFF;          /* Pure white text */
--foreground-muted: #A1A1A1;    /* Secondary text */
--foreground-subtle: #666666;   /* Tertiary/hint text */
```

## Typography

**Primary Font:** Satoshi (Variable)
- Headlines, body, UI elements
- Weights: 400 (regular), 500 (medium), 700 (bold)

**Mono Font:** Geist Mono
- Code, timestamps, technical info

## Voice & Tone

The Yellow Pill voice is:
- **Cool** — never try-hard, effortlessly confident
- **Mysterious** — suggests depth without over-explaining
- **Curt** — economical with words, every word earns its place
- **Confident** — assured, no hedging or qualifiers
- **Wise** — speaks from experience, not theory

This is **your best self** talking to you.

### Copy Examples

**Good:**
- "Are you ready?"
- "It's time."
- "Your feed. Your future."
- "We remember."

**Avoid:**
- Exclamation marks (rarely, if ever)
- "Welcome!" / "Hello!" / "Hey there!"
- Explanatory phrases like "This feature allows you to..."
- Buzzwords, jargon, filler

## Logo

Primary: `/public/images/assets/yellow_pill_logo.png`
- Tilted yellow capsule
- Use on pure black backgrounds
- Minimum size: 32px height

## Button Styles

**Primary (CTA):**
- Golden yellow gradient with 3D depth effect
- Rounded pill shape (full radius)
- Dark text on yellow

**Secondary:**
- Transparent with subtle border
- White text
- Hover: slight background

## Post Cards

- Dark card background (#111 or #0A0A0A)
- Circular poster avatars (yellow for now, gradients later)
- Verified checkmark next to poster names
- Subtle yellow glow on hover/focus (future)

## Spacing

- Generous whitespace
- Mobile-first: 16px side padding
- Card padding: 16-20px
- Between cards: 12-16px

## Animations

- Subtle and purposeful
- Fade-in on page load (0.3s ease)
- Slide-up for cards (staggered 0.05s delay)
- No bouncing, no playful animations
